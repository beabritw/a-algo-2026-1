## Análise de Complexidade:

O algoritmo é linear (O(n)) porque para calcular o fatorial de n
a função é chamada recursivamente exatamente n vezes.
porque, para uma entrada, a função realiza exatamente chamadas recursivas

Cada chamada realiza apenas operações simples (subtração e multiplicação) 
de tempo constante, o tempo total cresce de forma proporcional ao valor de n
